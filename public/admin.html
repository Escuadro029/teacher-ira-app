📬 auto email delivery flow

When a customer pays, your backend at /api/checkout verifies the Stripe payment, then emails all purchased ZIP files as attachments via Nodemailer or SendGrid. Admins can resend any order from the Orders page.

🗂️ adding your real files

Place ZIP files inside the /files/ folder at the project root. The filename must match exactly what is set in the Products panel — e.g. mental_health_schools.zip.
